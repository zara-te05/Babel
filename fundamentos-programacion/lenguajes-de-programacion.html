2. EL LENGUAJE DE LA MAQUINA

Un lenguaje de programación no es solo una herramienta técnica; es el traductor universal entre la idea de su programador y el procesador. Permite convertir ideas complejas en instrucciones precisas que la máquina puede ejecutar.

SECCIÓN 01

Categorías de Lenguaje

NIVEL_INFERIOR [LOW_LEVEL]

Interacción con el Hardware

Diseñados para hablar directamente con los componentes físicos. Ofrecen control total y velocidad máxima, pero requieren un conocimiento profundo de la arquitectura.

ENSAMBLADOR (ASM) precision_manufacturing
C / C++ settings_input_component
RUST settings_input_component
NIVEL_SUPERIOR [HIGH_LEVEL]

Abstracción y Legibilidad

Utilizan una sintaxis similar al inglés, facilitando la escritura y mantenimiento del software. Priorizan la productividad del desarrollador sobre el control de bajo nivel.

C# / JAVA psychology
PYTHON / KOTLIN language

Métodos de Procesamiento

inventory_2

COMPILADOS

El código completo se transforma en un archivo ejecutable antes de usarse. Es como traducir un libro entero antes de entregarlo al lector.

interpreter_mode

INTERPRETADOS

El código se traduce y ejecuta sobre la marcha, línea por línea. Funciona como un intérprete que traduce una conversación en tiempo real.
