03 · Características

          
Un entorno
completo.

        


        


          

            

              
            

            
Sintaxis

            
Resaltado completo para Re

            
Colores semánticos para keywords, tipos, strings, funciones, comentarios y puntuación — diseñados para enseñar la gramática del lenguaje, no solo embellecerlo.

          


          

            

              
            

            
Tipado

            
Verificación de tipos en tiempo real

            
El TypeChecker de Re analiza tu código mientras escribes. Los errores de tipo aparecen antes de ejecutar — aprendes el contrato del código desde el primer día.

          


          

            

              
            

            
Diseñador

            
Diseñador tipo Raptor integrado

            
Diseña tu algoritmo visualmente con bloques de proceso, decisión y entrada/salida. El sistema genera el código Re equivalente automáticamente.

          


          

            

              
            

            
Multi-archivo

            
Varios archivos en pestañas

            
Organiza tu proyecto en múltiples archivos .re — funciones, tipos, colecciones — y navega entre ellos sin perder el contexto.

          


          

            

              
            

            
Sin instalación

            
Funciona directo en el navegador

            
Logos corre completamente en el navegador. Sin instalar runtimes, sin configurar entornos. Abre la página y empieza a escribir código Re ahora mismo.

          


          

            

              
            

            
Exportar

            
Exporta tu código o diagrama

            
Descarga tus archivos .re o exporta tu diagrama de flujo como SVG para compartirlo, imprimirlo o usarlo en tu documentación.
