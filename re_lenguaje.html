Hola Mundo e Entrada de Datos
La sintaxis es natural. Todo programa vive dentro de un bloque program.
program HolaMundo { string nombre = input("¿Cuál es tu nombre? "); print("¡Hola, " + nombre + "!"); }
salida → ¡Hola, Sofía!
Tipos seguros y conversión explícita
A diferencia de Python, Re no mezcla tipos sin conversión. El TypeChecker te protege antes de ejecutar.
program TiposSeguros { int edad = 20; double altura = 1.75; bool activo = true; string mensaje = "Tengo " + to_str(edad) + " años."; print(mensaje); }
salida → Tengo 20 años.
Funciones con tipos explícitos
Los parámetros y el tipo de retorno se declaran. Aprende contratos de código desde el primer día.
int calcularArea(int base, int altura) { return (base * altura) / 2; } string saludar(string nombre) { return "Hola, " + nombre.upper(); } program Funciones { int area = calcularArea(10, 5); print("Área: " + to_str(area)); print(saludar("mundo")); }
salida → Área: 25  ·  Hola, MUNDO
Colecciones y bucle for-in
El bucle for-in infiere el tipo de la variable iteradora automáticamente.
program EjemploColecciones { // Cola tipada (FIFO) queue<int> turnos = [10, 20, 30]; turnos.enqueue(40); for t in turnos { print("Procesando: " + to_str(t * 2)); } }
salida → Procesando: 20, 40, 60, 80
Estructuras de control clásicas
While, do-while, if-else e intervalos con rangos 1..10. Control total, sin sorpresas.
program Logica { int contador = 5; while (contador > 0) { if (contador % 2 == 0) { print(to_str(contador) + " es par"); } else { print(to_str(contador) + " es impar"); } contador -= 1; } }
salida → 5 es impar, 4 es par, 3 es impar, 2 es par, 1 es impar

Sistema de tipos

Tipos y colecciones.

Re te da primitivos claros y colecciones genéricas listas para usar. Sin importar librerías, sin configuración extra.

Primitivos
int double string bool

Cuatro tipos base. Sin ambigüedad, sin conversión implícita. El compilador valida cada asignación antes de ejecutar.

int edad = 21; double pi = 3.14; string nombre = "Re"; bool activo = true; var x = 42; // tipado dinámico
Colección
list<T>

Listas de tamaño dinámico con métodos nativos. El tipo genérico garantiza que no mezcles elementos incompatibles.

list<string> nombres = ["Ana", "Luis"]; nombres.add("Sofía"); nombres.remove(0); string s = nombres.get(0);
.add() .remove() .get() .size()
Colección
queue<T>

Cola con disciplina FIFO. Aprende el patrón de colas que se usa en schedulers, sistemas de mensajería y servidores.

queue<int> fila = [1, 2, 3]; fila.enqueue(4); int primero = fila.dequeue(); // → 1 print(to_str(fila.size()));
.enqueue() .dequeue() .size()
Colección
stack<T> array<T>

Pila LIFO para recursión y parsing. Arreglo de tamaño fijo para memoria predecible.

stack<string> historial = []; historial.push("inicio"); historial.push("página1"); string ult = historial.pop(); // → "página1" array<int> nums = [1,2,3]; // tamaño fijo
.push() .pop() .size() .get()

Instalación

Listo en dos comandos.

Requiere Node.js 18 o superior. El instalador compila el proyecto y registra los comandos re y re-lsp globalmente.

Linux
$ chmod +x install.sh
$ ./install.sh

Crea enlaces simbólicos en /usr/local/bin/re

macOS
$ chmod +x install.sh
$ ./install.sh

Requiere Node.js ≥ 18

Windows
PS> Set-ExecutionPolicy Bypass
PS> .\install.ps1

Ejecutar PowerShell como Administrador

LSP + Editores

Tu editor, con superpoderes Re.

El instalador expone re-lsp, un servidor de lenguaje completo. Cualquier editor compatible con LSP obtiene autocompletado y validación en tiempo real.

Integración mediante el plugin LSP oficial de Sublime. Configuración en un solo archivo.
